Akka-HTTP and SSL connections

2016-03-06

For a rest-based project I wanted to create an akka-http server with SSL-support. Unfortunately I found only some scattered information, e.g. on Stackoverflow. hence I present this very short but complete tutorial.

First, we generate a self-signed certificate with

keytool -genkey -keyalg RSA -keystore ssl-test-keystore.jks -storepass abcdef -validity 360 -keysize 2048

which creates a file ssl-test-keystore.jks which should be put on the classpath.
